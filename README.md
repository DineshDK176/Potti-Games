# Potti Games - Game Store

## Overview

Potti Games is a modern game store web application built with Next.js. Browse, discover, and explore a wide variety of games with features including:

- Game browsing with categories and filters
- Trending and featured games sections
- Shopping cart functionality
- Wishlist management
- User profiles with local storage
- Responsive design for all devices

This repository stays in sync with your deployed chats on [v0.app](https://v0.app).
